# 使用技術
- Ruby 2.6.3
- Ruby on Rails 5.1.7
- postgreSQL 12
- Docker
- GitHub
- Nginx + Puma


# 機能一覧、使用したgemなど
- APIによる簡単ログイン(twitterのみ)
- 投稿機能
- コメント機能(Ajax)
- ユーザー登録機能、ログイン機能(devise)
- ページネーション機能(kaminari)
- いいね機能(Ajax)
- フォロー機能(Ajax)
- 検索機能(Ransack)

# テスト
